Cargar HistropediaJS y primera timeline

Bloques 1–3

HTML mínimo

Importa dependencias y crea la timeline vacía.

<div id="timeline"></div>

<script src="https://code.jquery.com/jquery-3.5.1.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery-mousewheel/3.1.13/jquery.mousewheel.min.js"></script>
<script src="https://cdn.histropedia.com/histropedia-1.2.0.min.js"></script>

<script>
  const container = document.getElementById('timeline');
  const timeline = new Histropedia.Timeline(container, {
    width: container.clientWidth,
    height: 480
    // zoom/style/article: se ajustan luego
  });
</script>
  • jQuery + mousewheel = dependencias del canvas de Histropedia.
  • El objeto timeline dibuja sobre el div y queda listo para recibir datos.

¿Qué es un article?

Histropedia pinta tarjetas a partir de objetos con título, fecha y opcionalmente imagen.

const articles = [
  {
    id: 1,
    title: "Nombre del evento u obra",
    subtitle: "Texto pequeño",
    from: { year: 1990, month: 1, day: 1, precision: Histropedia.PRECISION_YEAR },
    to: null,
    imageUrl: "https://ejemplo.com/imagen.jpg"
  }
];

timeline.load(articles);
  • title y subtitle = texto visible.
  • from/to aceptan solo año o fecha completa (PRECISION_YEAR si solo hay año).
  • imageUrl recorta la imagen dentro de la tarjeta.

Mini timeline de 3 eventos

Para mostrar el efecto inmediato antes de hablar de estilos.

const demoEvents = [
  { id: 1, title: "Evento A", subtitle: "Ejemplo", from: { year: 1890, precision: Histropedia.PRECISION_YEAR }, to: null, imageUrl: null },
  { id: 2, title: "Evento B", subtitle: "Ejemplo", from: { year: 1920, precision: Histropedia.PRECISION_YEAR }, to: null, imageUrl: null },
  { id: 3, title: "Evento C", subtitle: "Ejemplo", from: { year: 1975, precision: Histropedia.PRECISION_YEAR }, to: null, imageUrl: null }
];

timeline.load(demoEvents);
  • Muestra tres tarjetas sin depender de SPARQL ni de imágenes.
  • Luego reemplaza demoEvents por el JSON de la consulta preprocesada.

Decoración de tarjetas y eje

Bloque 4

Idea general

En cada HTML hay estilos configurados así:

const timeline = new Histropedia.Timeline(container, {
  // ...
  article: {
    defaultStyle: ARTICLE_STYLE,
    defaultActiveStyle: ACTIVE_STYLE,
    autoStacking: { /* ... */ }
  },
  style: { marker: { /* eje */ }, dateLabel: { /* labels */ } }
});
  • defaultStyle = tarjeta normal; defaultActiveStyle = tarjeta al hacer click.
  • Con pocos cambios (color, width, borde, sombra, radios) cambias totalmente la personalidad.
  • style.marker y style.dateLabel controlan el eje y las marcas de fecha.

Cómo mostrar los ejemplos sin marear

A

Presenta la arquitectura, no el SPARQL

Cuenta que el JSON ya viene listo de Wikidata. En código, resalta dónde mapeas cada fila a title, subtitle, fechas e imagen.

B

Enfócate en ARTICLE_STYLE y ACTIVE_STYLE

En DevTools o editor, marca solo esos bloques. Cambia color/base/width/sombra, recarga y compara densidad.

C

Muestra el eje al tocar zoom

Toca style.marker y style.dateLabel para que vean cómo reaccionan las marcas de tiempo.

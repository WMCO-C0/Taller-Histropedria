Tips de uso durante el taller

A

Probar en Wikidata Query Service

Abre el endpoint, pega la query de cada demo, ejecuta y revisa columnas. Exporta a JSON y compara con el mapeo en el JS.

B

Modificar filtros

Juega con P39 (cargo), P27/P495 (pais), P577 (fecha), P18 (imagen) y LIMIT. Observa como cambia la linea de tiempo.

C

Crear una copia rapida

Duplica un index en una carpeta nueva, cambia la query y los campos de mapeo (titulo, subtitulo, fecha from/to, imagen) para tu propio tema.
